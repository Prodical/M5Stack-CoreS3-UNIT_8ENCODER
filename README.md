# M5Stack CoreS3 UNIT_8ENCODER Arduino Sketch

Arduino sketch for testing the 8-encoder unit with M5Stack CoreS3.

## Hardware
- M5Stack CoreS3
- UNIT_8ENCODER module

## Files
- `unit_8encoder_testing_Lewis_8.ino` - Main sketch
- `DIN_Condensed_Bold30pt7b.h` - Font file for display

## Setup Instructions

1. Install M5Stack Arduino libraries:
   - Install via Arduino IDE Library Manager
   - Board: M5Stack CoreS3

2. Upload sketch to M5Stack CoreS3
